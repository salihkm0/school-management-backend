# School Management System Backend

A comprehensive school management system backend built with Node.js, Express, MongoDB, and Socket.IO.

## Features

- 🔐 JWT Authentication with Refresh Tokens
- 👥 Role-based Access Control (Admin, Staff, Parent)
- 📊 Real-time Notifications with Socket.IO
- 📁 CSV/Excel Import/Export
- 📄 PDF Report Card Generation
- 📧 Email Notifications
- 🚀 Redis Caching
- 📈 Analytics and Reports
- 🏫 Complete CRUD Operations
  - Student Management
  - Staff Management
  - Class Management
  - Exam Management
  - Marks Entry
  - Attendance Tracking
  - Staff Duty Assignment

## Prerequisites

- Node.js 16+
- MongoDB 6+
- Redis 7+
- npm or yarn
